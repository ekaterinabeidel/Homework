В Java enum (перечисление) является типом данных, который представляет собой набор предопределенных констант.
Он используется для создания переменных,
которые могут принимать только одно из заданных значений, которые перечислены в перечислении.

Преимущества использования перечислений:

Ограничение значений:
Перечисления обеспечивают строгую типизацию и гарантируют,
что переменная может принимать только одно из определенных значений.

Читаемость кода:
Перечисления делают код более читаемым и понятным,
поскольку они представляют собой логически связанный набор констант.

Безопасность:
Использование перечислений обеспечивает безопасность типов, предотвращая ошибки ввода данных.


Методы, которые можно использовать с перечислениями, включают:

values():
Этот метод возвращает массив, содержащий все константы перечисления в том порядке, в котором они были объявлены.
Например:
Country[] countries = Country.values();

valueOf(String name):
Этот метод возвращает константу перечисления по ее имени.
Например:
Country country = Country.valueOf("USA");

name():
Этот метод возвращает имя константы перечисления в виде строки.
Например:
String countryName = Country.USA.name();
Дополнительно, вы можете добавлять собственные методы и поля в перечисления, как и в обычные классы.
