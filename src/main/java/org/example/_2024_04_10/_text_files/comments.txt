
Country.values():
Мы используем метод values() для перечисления Country, чтобы получить массив всех значений перечисления Country.

random.nextInt(Country.values().length):
Мы вызываем метод nextInt(int bound) у объекта random, чтобы получить случайное целое число
от 0 до длины массива значений перечисления Country. Это число используется для выбора случайного индекса из массива.

Country.values()[random.nextInt(Country.values().length)]:
Мы используем случайный индекс, полученный в предыдущем шаге,
чтобы получить случайное значение из массива значений перечисления Country.

"Street " + random.nextInt(100):
Мы используем метод nextInt(int bound) у объекта random,
чтобы получить случайное целое число от 0 до 99,
а затем добавляем это число к строке "Street " для создания случайной улицы.

random.nextInt(201), // Случайный номер дома от 0 до 200
random.nextInt(50) // Случайный номер квартиры от 0 до 49
